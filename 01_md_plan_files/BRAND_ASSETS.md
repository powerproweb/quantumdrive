# BRAND_ASSETS.md — QuantumDrive.io

Brand identity reference for QuantumDrive.io.
Last updated: 2026-04-14

---

## Brand Essence

**Name:** QuantumDrive.io
**Tagline:** Sovereign Systems Portfolio
**One-line pitch:** The front door for 10+ projects built so everyone in the flow wins.
**Brand promise:** Honest, transparent portfolio presentation — every project shown as it really is.

### Brand Values

1. **Honesty** — No inflated claims, no fake status badges
2. **Clarity** — Every project understood in seconds
3. **Modularity** — Add a project by editing one JSON file
4. **Sovereign** — Every party in the flow wins

---

## Tone of Voice

QuantumDrive's voice is **confident**, **precise**, and **direct**.

It always sounds like:
- Someone who knows what they're building and why
- A premium portfolio that respects the visitor's intelligence
- A Sovereign System — honest about what exists and what's coming

### Sovereign Brand Rules

- **No manufactured urgency**
- **No obscured value exchange**
- **No blame-shifting language**
- **No false social proof** — every status badge is real
- **Accessibility is not optional**

See `01_md_plan_files/SOVEREIGN.md` for the full doctrine.

## Logo

The logo is a CSS-generated gradient orb (purple/cyan radial gradients on a rounded square). No external image file required — it is rendered by the `.logo` class in `assets/css/styles.css`.
